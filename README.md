# wildbean
## 目标
面向移动的响应式布局，H5技术，流畅迅捷的交互体验设计

## RoadMap

1. v0.1: 展示界面的藏书查询，自助借阅
1. v0.2: 管理界面的 Dashboard, 图书管理，借阅管理，会员管理
1. v0.3: 完善展示界面
2. v0.4: 完善管理各模块
3. v0.5: 插件化，模块化设计
4. v0.6: 编写单元测试和集成测试
5. v0.7: 自动化产品构建流程
6. v0.8: CI
7. v0.9: 跨终端，跨浏览器兼容
8. v1.0: 总结测试发布

### 管理

####  Dashboard

* 藏书情况（馆藏图书多少，分类）
* 借阅情况（总借出多少图书，每一本是谁借的，超过长时间没有归还的，图书总借阅次数统计）
* 会员情况（会员总数，快到期会员报警，活跃的，会员借阅情况，分时段查看会员加入情况）
* 收入情况（记账系统，会员收入，每月支出，押金情况）
